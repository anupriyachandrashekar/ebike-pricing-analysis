# E-Bike Pricing Strategy & Transportation Analysis

## Overview

This project analyzes transportation behavior using survey data to understand user preferences and determine optimal pricing strategies for e-bike services.


## Objectives

* Analyze user travel behavior and preferences
* Evaluate pricing sensitivity across different user segments
* Identify an optimal pricing strategy


## Tools & Technologies

* R (statistical modeling)
* Excel (data handling)
* Data visualization techniques


## Key Work

* Processed and analyzed survey data
* Applied statistical models to evaluate demand elasticity
* Segmented users based on behavior and pricing response
* Compared multiple pricing scenarios


## Results

* Identified optimal pricing strategy based on demand patterns
* Provided insights into user willingness to pay
* Supported data-driven transportation planning
